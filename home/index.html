Projeto MarketPlace de Filmes

Com o objetivo de criar uma experincia de compra mais eficiente e dinmica, mergulhei de cabea na implementao de diversas tecnologias de ponta.

  • Integrao de API Rest: Utilizei requisies HTTP para consumir uma API Rest, proporcionando uma conexo instantnea com um vasto leque de produtos e informaes atualizadas em tempo real.
  • Snackbar Dinmico: Ao adicionar ou remover produtos do carrinho, implementei um snackbar que oferece feedback imediato ao usurio, tornando a experincia de compra mais interativa e satisfatria.
  • Angular 17 em Ao: Aproveitei ao mximo as novas funcionalidades e melhorias do Angular 17, incluindo o uso de sinais (signals) para otimizar o fluxo de dados e aprimorar a performance da aplicao.
  • Layout Responsivo de Ponta a Ponta: Um dos pontos-chave do projeto foi o desenvolvimento de um layout totalmente responsivo, garantindo uma experincia consistente e funcional em todos os dispositivos, desde smartphones at desktops de alta resoluo.

Estou incrivelmente empolgado com os resultados alcanados e animado para compartilhar mais detalhes em breve. Este projeto apenas o comeo de uma jornada contnua de inovao e aprimoramento na experincia de compra online.
